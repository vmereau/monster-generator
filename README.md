## Description

Experimental project using Gemini 2.0 and stable diffusion to generate monsters based on user input
using [Nest](https://github.com/nestjs/nest) framework

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
